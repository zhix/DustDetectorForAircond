# DustDetectorForAircond
A student's project, publishing it for future reference

## How it functions?
1) 5 LDR --> Arduino --> Tri-color LED 
2) LED indicators: 
* Red = Dirtiest Filter
* Blue = Dusty Filter
* Green = Clean Filter

## Future work: 
* To automate the determination of light threshold
